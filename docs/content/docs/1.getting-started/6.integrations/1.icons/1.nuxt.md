## Usage

They are very easy to use:

```vue [SomeComponent.vue]

import Bitrix24Icon from '@bitrix24/b24icons-vue/common-service/Bitrix24Icon'
</script>

<template>
  <div class="rounded ...">
    <Bitrix24Icon class="size-15 ..." />
  </div>
</template>
```
