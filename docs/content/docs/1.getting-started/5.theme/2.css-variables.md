## Text

Bitrix24 UI provides Tailwind CSS utility classes for text colors, allowing you to use class names like `text-legend` or `text-description`:

::code-preview{class="[&>div>p]:flex [&>div>p]:flex-wrap [&>div>p]:gap-2"}
[Dimmed]{class="text-dimmed text-sm px-4 py-1.5 inline-block rounded-md"}
[Muted]{class="text-muted text-sm px-4 py-1.5 inline-block rounded-md"}
[Description]{class="text-description text-sm px-4 py-1.5 inline-block rounded-md"}
[Legend]{class="text-legend text-sm px-4 py-1.5 inline-block rounded-md"}
[Label]{class="text-label text-sm px-4 py-1.5 inline-block rounded-md"}

#code
```vue
<template>
  <span class="text-dimmed">Dimmed</span>
  <span class="text-muted">Muted</span>
  <span class="text-description">Description</span>
  <span class="text-legend">Legend</span>
  <span class="text-label">Label</span>
</template>
```

## Body

Bitrix24 UI applies default classes on the `<body>`{lang="html"} element of your app for consistent theming across light, edge-light, edge-dark and dark modes:

```css
body {
  scrollbar-gutter: stable;
  background: var(--air-theme-background);
  @apply antialiased font-(family-name:--ui-font-family-system) text-(--b24ui-typography-legend-color) scheme-light dark:scheme-dark edge-light:scheme-light edge-dark:scheme-light
}
```
