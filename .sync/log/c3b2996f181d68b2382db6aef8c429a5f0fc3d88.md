## Upstream change
`FileUpload.vue`: the non-`button` variant renders a `role="button"` `<div>`
which can't use the native `disabled` attribute, so a disabled dropzone was not
announced. Adds `:aria-disabled="variant === 'button' ? undefined : (disabled || undefined)"`
on the base element (native `disabled` still used for the `button` variant).
