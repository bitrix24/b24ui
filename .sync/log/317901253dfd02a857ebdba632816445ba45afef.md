# Port: fix(theme): use logical properties for RTL

**Upstream:** `317901253dfd02a857ebdba632816445ba45afef` (nuxt/ui, #6724)
**Decision:** port — theme (subset; absent components skipped)

## Upstream change
Converts physical CSS directional utilities to their logical equivalents across
16 theme files so components lay out correctly under RTL by default:
`left`→`start`, `right`→`end`, `text-left/right`→`text-start/end`,
`rounded-l`→`rounded-s`, `rounded-tl/tr/bl/br`→`rounded-ss/se/es/ee`,
`transition-[left,right]`→`transition-[inset-inline-start,inset-inline-end]`,
plus explicit `rtl:` counterparts for `translate-x` transforms and
`cursor-*-resize` handles that can't be expressed logically. Also adds an RTL
guidance section to `.github/contributing/theme-structure.md` and `AGENTS.md`.

## b24ui port
b24ui has 9 of the 16 touched theme files; applied the logical-property swap to
the 8 that actually carry the physical utility (prose/tr is a no-op stub):

- **`navigation-menu.ts`** — `viewportWrapper` `left-0`→`start-0` (b24ui's
  `content` slot is empty, so upstream's 2nd hunk has no analog).
- **`prose/callout.ts`** / **`prose/card.ts`** — external-icon `right-2`→`end-2`.
- **`prose/pre.ts`** — copy button `right-[11px]`→`end-[11px]`.
- **`prose/td.ts`** / **`prose/th.ts`** — align `text-left/right`→`text-start/end`.
- **`sidebar.ts`** — all 6 hunks: `transition-[left,right,width]`→logical,
  `left-0`/`right-0` container → `start-0`/`end-0`, rail `translate-x` gains
  `rtl:` counterparts, `cursor-*-resize` gains `rtl:` + collapsed `rtl:`
  variants, `-left-/-right-(--sidebar-width)`→`-start-/-end-`.
- **`table.ts`** — header `text-left rtl:text-right`→`text-start`; loading
  separator `left-0`→`start-0`.
